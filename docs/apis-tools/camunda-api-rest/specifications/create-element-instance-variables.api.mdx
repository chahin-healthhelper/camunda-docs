<h1 className={"openapi__heading"}>Update element instance variables</h1>

<MethodEndpoint
  method={"post"}
  path={"/element-instances/{elementInstanceKey}/variables"}
></MethodEndpoint>

Updates all the variables of a particular scope (for example, process instance, flow element instance) with the given variable data.
Specify the element instance in the `elementInstanceKey` parameter.
